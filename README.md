## Development

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=johnsoncodehk.volar)

Setup:

```js
npm install
```

Start dev server:

```sh
npm run netlify
```

Build:

```sh
npm run build
```

Preview build:

```sh
npm run serve
```
